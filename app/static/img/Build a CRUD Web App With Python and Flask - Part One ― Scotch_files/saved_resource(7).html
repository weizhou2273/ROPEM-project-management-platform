              @mbithenzomo:disqus Great tutorial. I find that point 2 of your description of what you have in your config.py is not entirely accurate. Setting Debug = True in the config file does not automatically enable the reloading of the server when edits are made. I've tried it with python 2.7.X and 3.5.X with same result. As seen in this issue: https://github.com/pallets/flask/issues/1979 , FLASK_DEBUG environment variable must be set to 1 for reloading to work. If you're able to get reloading to work with the new flask cli by setting Debug = True in configuration files only, please share how you got that to work. I'll rather not go back to the old way of doing app.run (Debug = True) if possible only to get reloading to work properly.
